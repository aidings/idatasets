# idatasets

#### 介绍
idatasets

#### 安装教程
`pip install git+https://gitee.com/cvding/idatasets.git`

#### 使用说明
1. augment模块：图片增强，视频帧增强
2. core模块：细粒度dataset，近距离dataset
3. utils模块：图像读取，列表读取，lmdb读取，pool列表读取
